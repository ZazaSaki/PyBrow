Get not included values of a given worksheet from a given list of values and also filter the not included values from the intern database.

Parameters
profilesthe list to filter
sheetworksheet to filter with
colcolumn to start to insert the values (default value = 1, start counting on 1, ids are unique)
offsetindex in the <profiles>, that has the ids to compare (deafault value = 0, start counting on 0)
Returns
List of filtered values

Assuming we have the next worksheet named \a "sheet" from a spreadsheet named \a "spread":

Aplaying this code :

list_to_insert = [
['row1', 'name', 'id1', 'age'],
['row2', 'name', 'id2', 'age'],
['row3', 'name', 'id3', 'age']
]
spreadsheet = client_session.open("spread")
worksheet = spreadsheet.worksheet("sheet")
#utilization
getNotIncludedByCol(list_to_insert, worksheet, offset = 2)

the output would be:

[
['row1', 'name', 'id1', 'age'],
['row3', 'name', 'id3', 'age']
]

Unless the data base already had the "id1" and "id3", in that case the contained ids in data base would be removed.

DataManager.filterList
